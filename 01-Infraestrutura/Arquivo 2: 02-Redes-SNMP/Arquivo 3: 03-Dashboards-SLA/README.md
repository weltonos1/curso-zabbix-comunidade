# 📊 Módulo 3: Inteligência de Dados e Gestão de SLA

Onde os dados se tornam decisões de negócio. O foco aqui é o gestor e o time de NOC.

## 📈 1. NOC de Elite no Grafana
- Dashboards com Variáveis Dinâmicas.
- Uso de Painéis de **Stat** e **Gauge** para visibilidade imediata.
- Integração de Geomapas para infraestruturas distribuídas.

## 🤝 2. Gestão de SLA e SLO
- Configuração de Árvore de Serviços no Zabbix 7.0.
- Cálculo de disponibilidade mensal.
- Relatórios automáticos de tempo de inatividade (Downtime).

## 🛠️ Atividade Prática
1. Integrar Zabbix ao Grafana via API.
2. Criar um Serviço de "Link de Internet" com SLA de 99.9%.
3. Configurar Alerta no Telegram com gráfico de latência anexado.
